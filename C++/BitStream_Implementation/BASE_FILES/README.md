# BitStream

Your task is to rewrite the given reference implementation for a BitStream class
and resolve any issues you find.

Your implementation should:

- avoid loops within  `read` and `write` methods,
- improve the underlying memory model,
- handle container growth correctly.

Both implementations will be discussed during the interview.
